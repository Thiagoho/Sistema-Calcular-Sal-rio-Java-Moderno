# Sistema de Cálculo de Salário - Java Moderno

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais e modernos da linguagem Java, simulando um sistema simples de cálculo de salário de funcionários.

A aplicação permite cadastrar um funcionário, calcular seu bônus de acordo com o cargo e exibir um relatório completo contendo salário base, bônus e salário final.

## Tecnologias Utilizadas

* Java 21
* Record
* Var
* Switch Expression
* Métodos
* Text Block

## Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados os seguintes conceitos:

* Criação e utilização de métodos
* Programação Orientada a Objetos
* Uso de Records para modelagem de dados
* Utilização de `var` para inferência de tipos
* Implementação de `switch expression`
* Uso de Text Blocks para exibição de mensagens formatadas
* Cálculos com retorno de métodos
* Organização e separação de responsabilidades no código

## Funcionalidades

* Cadastro de funcionário
* Definição do cargo
* Cálculo automático de bônus
* Cálculo do salário final
* Exibição de relatório completo

## Regras de Negócio

### Gerente

* Bônus fixo de R$ 1.500,00

### Vendedor

* Bônus de 10% sobre o salário base

### Estagiário

* Bônus fixo de R$ 300,00
